# MyZomato 
MyZomato is a food delivery app built in Kotlin and a RoomDB clone of Zomato. Users can explore restaurants' menus and add them into carts.

# Features
- Restorants
- Menu
- Cart
- Order History

## Library Used
- RoomDB
- Sticky ScrollView
- Country Code Picker
- Live Data
